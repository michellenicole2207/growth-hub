## Notes Convention

Each chapter `notes.md` includes:

- **Big idea** — the core concept of the chapter in plain language
- **Cognitive levels** — K1/K2/K3 legend and the official learning objectives (LOs) for that chapter
- **Content sections** — each section labeled with 🧠 Remember (K1), 💡 Understand (K2), or ⚙️ Apply (K3)
- **Key Terms Cheat Sheet** — all terms to memorize (🧠 K1)
- **Practice questions** — exam-style questions at the end
